Want full verification?

This browser demo validates URLs only (HTTP reachability). Full verification — author order, year accuracy, venue mismatch, title hallucination detection, and DOI cross-check against 10 academic databases — requires the CLI tool, which runs natively with async networking, rate limiting, and complex author comparison algorithms that can't run in a browser.

pip install -r requirements.txt
python3 main.py your_references.bib --json -o report.json -v

BibEval validates against dblp, Semantic Scholar, Crossref, arXiv, ACL, NeurIPS, CVPR, ACM, OpenAlex, and OpenReview — all free, no API keys needed.
